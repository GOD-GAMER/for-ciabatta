# Feature Toggles

Enable/disable commands and features from the dashboard.

## Open Toggles Tab
- Dashboard ? Toggles tab
- Flip switches for features
- Click Save Toggles

## Groups
- commands.__all__ – toggle all commands
- games.__all__ – toggle all games
- economy.__all__ – toggle economy features

## Specific Toggles
- commands.guess – disable !guess
- games.bread_fights – disable fights
- economy.gifting – disable !gift

Changes apply immediately; no restart required.
